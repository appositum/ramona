### Setup
```bash
$ export TOKEN="discord bot token"
$ git clone https://github.com/appositum/ramona.git
$ cd ramona/
$ mix deps.get && mix run --no-halt
# or run in interactive mode
$ mix deps.get && iex -S mix
```
